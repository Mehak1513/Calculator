
🧮 Mehak Calculator (Python Mini Project)

📌 Project Description

This is a simple menu-driven calculator program written in Python.
The program takes two numbers from the user and performs basic arithmetic operations based on the option selected by the user.

🚀 Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Menu based user selection
- Simple command line interface

🛠️ Technologies Used

- Python (Basic Programming)
- Command Line Interface

📂 Program Logic

1. The program welcomes the user.
2. It asks the user to enter two numbers.
3. A menu of operations is displayed:
   - Addition
   - Subtraction
   - Multiplication
   - Division
4. The user selects an option.
5. The program performs the selected operation and displays the result.

▶️ How to Run the Program

1. Install Python on your computer.
2. Download or clone this repository.
3. Open terminal or command prompt.
4. Run the program using:

python calculator.py

5. Enter two numbers and select the operation.
